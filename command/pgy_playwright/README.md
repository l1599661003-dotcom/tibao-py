# 蒲公英邀约表单批量发送工具

## 功能说明

从Excel文件导入蒲公英链接，提取博主ID，批量发送邀约表单。

## 使用方法

### 1. 配置设置

修改 `config.ini` 文件中的产品信息：

```ini
[FORM_CONFIG]
# 邀约表单配置
product_name = 你的产品名称
product_type_description = 你的产品类型描述
```

### 2. 准备Excel文件

创建一个Excel文件，包含以下列：
- **蒲公英链接**（必填）：包含蒲公英博主详情页的链接

### 3. 运行程序

```bash
python send_pgy_yaoyue.py
```

程序会：
1. 弹出Excel文件选择对话框
2. 加载Excel数据
3. 初始化浏览器并登录
4. 遍历Excel中的每一行
5. 从蒲公英链接中提取ID
6. 为每个ID发送邀约表单

## 工作流程

1. **选择Excel文件**：程序启动后弹出文件选择对话框
2. **登录蒲公英**：自动打开浏览器，等待用户登录
3. **批量处理**：
   - 从Excel读取蒲公英链接
   - 提取博主ID
   - 访问邀约表单页面
   - 复用上次填写内容
   - 填写产品名称和合作内容介绍
   - 点击确认发起邀约
4. **完成处理**：所有行处理完成后自动关闭

## 注意事项

- 确保Excel文件包含"蒲公英链接"列
- 程序会自动复用上次填写的表单内容
- 每个邀约之间会有5-10秒的延迟
- 需要手动登录蒲公英账号
