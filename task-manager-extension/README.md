# 📋 工作清单管理插件

一个专业的工作清单管理浏览器插件，支持日、周、月三种视图的任务管理，帮助用户高效管理日常工作任务。

## ✨ 核心功能

### 🎯 三种视图管理

**日视图**：
- 显示选定日期的任务清单
- 支持添加、勾选完成、删除任务
- 实时显示今日/当日任务数量
- 左右箭头切换日期

**周视图**：
- 显示当月的1-4周任务管理
- 每周独立的任务列表和勾选框
- 2x2网格布局，清晰展示4周进度
- 按月份导航

**月视图**：
- 显示整月的任务目标清单
- 月度任务的添加和完成管理
- 长期目标和计划追踪

### ✅ 任务管理功能

- **添加任务**：输入框 + 按钮，支持回车快捷键
- **勾选完成**：每个任务都有复选框，点击切换完成状态
- **删除任务**：鼠标悬停显示删除按钮，确认删除
- **实时统计**：显示总任务数、已完成数、完成率

### 💾 数据持久化

- 使用Chrome Storage API本地存储
- 支持跨浏览器会话数据保持
- 按日期、周、月智能分类存储
- 数据导入导出功能（可选）

## 🎨 界面设计

### MetaMask风格界面
- **尺寸**：480x600像素，适配桌面和移动设备
- **毛玻璃效果**：现代化背景虚化和透明度设计
- **渐变背景**：优雅的多色渐变背景
- **动画交互**：流畅的hover和点击动画效果

### 响应式布局
- 周视图采用2x2网格布局
- 移动端自适应为单列布局
- 任务列表支持滚动显示
- 清晰的视觉层次和间距设计

## 📖 使用指南

### 基本操作

1. **切换视图**
   - 点击顶部"日视图"、"周视图"、"月视图"按钮
   - 每个视图独立管理对应时间段的任务

2. **添加任务**
   - 在对应视图的输入框中输入任务内容
   - 点击"+"按钮或按回车键添加
   - 任务会自动保存到对应的时间段

3. **管理任务**
   - **完成任务**：点击任务前的复选框标记完成
   - **删除任务**：鼠标悬停显示"×"按钮，点击删除
   - **查看状态**：已完成任务会显示删除线和半透明效果

4. **日期导航**
   - **日视图**：使用左右箭头按天切换
   - **周视图/月视图**：使用左右箭头按月切换
   - 日期信息实时显示在导航栏

5. **数据统计**
   - 底部状态栏显示总任务数、已完成数、完成率
   - 各视图头部显示对应时间段的任务数量

## 🚀 安装方法

### Chrome浏览器
1. 打开Chrome浏览器
2. 进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择插件文件夹 `task-manager-extension`
6. 插件安装完成

### Firefox浏览器
1. 打开Firefox浏览器
2. 进入 `about:debugging#/runtime/this-firefox`
3. 点击"临时载入附加组件"
4. 选择 `manifest.json` 文件
5. 插件安装完成

### 使用插件
- 安装完成后，点击浏览器工具栏中的插件图标
- 首次使用会显示空的任务列表
- 开始添加任务，体验高效的任务管理
